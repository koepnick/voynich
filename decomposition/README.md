# Decomposition

This is simply a theory that I cannot get out of my head. So I'm giving it its own directory.

The idea is simple. This, and the fact that I am **not** a linguist should raise red flags for everybody reading this.

I believe that each VM glyph can be decomposed into a set of simpler glyphs. 

Even if this is true, it will probably never be provable and may not have any effect on translation efforts. But this is my hobby and I wanna do it. 

I believe that the structure of the glyphs is a grid-like pattern with sub-grids able to live within a standard grid.
